## Usage Example

```python
from spark_cube.core.minimal_spark import MinimalSparkCube, Signal, Intention, SignalType

# Initialize cube (zero inherited knowledge)
cube = MinimalSparkCube()

# Set intention (what we want to manifest)
intention = Intention(
    desired_qualities=["pattern", "analysis"],
    desired_form="analytical_response",
    clarity=0.9,
    energy=0.8
)

# Process with reflection
signal = Signal(SignalType.PATTERN, [1, 2, 3, 4, 5], 0.7)
result = cube.process_with_reflection(signal, intention)

# Examine coherence
print(f"Overall Coherence: {result['overall_coherence']:.3f}")
print(f"Path Chosen: {result['path']}")
print(f"Amplification: {result['amplification']}x")

# View per-node evaluations
for node, score in result['dimensional_scores'].items():
    print(f"  {node}: {score:.3f}")

# Over time, cube learns which pathways produce coherent outcomes
# High coherence → ALIGNMENT → 1.5x amplification → strengthened
# Low coherence → DIVERSION → 0.7x amplification → suppressed
```
